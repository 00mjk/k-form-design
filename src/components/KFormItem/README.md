KFormItem
====
构建表单控件
根据参数不同的type，生成对应的控件


### 使用方式

```
# 参数record：
		{
			"type": "input",
			"name": "单行文本",
			"options": {
				"width": "100%",
				"defaultValue": "",
				"placeholder": "请输入",
				"disabled": false
			},
			"model": "input_1569051592984",
			"key": "input_1569051592984",
			"rules": [
				{
					"required": false,
					"message": "必填项"
				}
			]
		}
	
```
